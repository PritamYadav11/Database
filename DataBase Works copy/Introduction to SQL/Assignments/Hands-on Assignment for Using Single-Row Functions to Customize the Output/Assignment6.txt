Q. Display the employee name and calculate the number of months between today and the hire date. Label the column MONTHS_WORKED. Round the result to the nearest whole number and order by months employed.

Answer :- 

SELECT ENAME,
       ROUND(MONTHS_BETWEEN(SYSDATE, HIREDATE)) AS MONTHS_WORKED
FROM EMP
ORDER BY MONTHS_WORKED;
